# 🚀 StagePro

[cite_start]**StagePro** est une plateforme web moderne conçue pour simplifier la recherche de stages pour les étudiants et la gestion des candidatures pour les pilotes et administrateurs. [cite: 1, 33]

---

## ✨ Fonctionnalités principales

### 🎓 Pour les Étudiants
* [cite_start]**Recherche dynamique** : Un formulaire de recherche intuitif pour trouver des offres par mots-clés. [cite: 1, 2]
* [cite_start]**Gestion de Wish-list** : Enregistrez vos offres favorites pour les retrouver plus tard. [cite: 42, 103]
* [cite_start]**Suivi des candidatures** : Espace dédié pour consulter l'état de vos postulations en temps réel. [cite: 34, 43]

### 👨‍✈️ Pour les Pilotes & Administrateurs
* [cite_start]**Espace Pilote** : Accompagnement des étudiants et suivi des offres. [cite: 34]
* [cite_start]**Espace Admin** : Gestion complète des comptes, des entreprises et administration globale du site. [cite: 35]

---

## 🛠️ Aspect Technique & Sécurité

* [cite_start]**Architecture MVC** : Développé en PHP avec une séparation claire entre les contrôleurs, les modèles et les vues Twig. 
* [cite_start]**Sécurité HTTPS** : Configuration optimisée pour le protocole SSL (gestion des certificats locaux via XAMPP). [cite: 46, 56]
* [cite_start]**Gestion des Cookies (RGPD)** : Bannière de consentement personnalisable avec préférences pour les statistiques. [cite: 84, 88]
* [cite_start]**FAQ Interactive** : Section d'aide intégrée pour répondre aux questions courantes (éligibilité, gratuité, etc.). [cite: 40, 45]

---
