🚀 StageProStagePro est une plateforme web moderne conçue pour simplifier la recherche de stages pour les étudiants et la gestion des candidatures pour les pilotes et administrateurs. Développé avec une architecture PHP propre et le moteur de template Twig, le site offre une expérience utilisateur fluide et sécurisée.✨ Fonctionnalités🎓 Pour les ÉtudiantsRecherche d'offres : Système de recherche dynamique pour trouver le stage idéal.Gestion de Wish-list : Possibilité d'ajouter et de retirer des offres en favoris pour les consulter plus tard.Suivi des candidatures : Espace dédié pour visualiser l'état de ses postulations.👨‍✈️ Pour les Pilotes & AdminsTableaux de bord dédiés : Interfaces spécifiques pour la gestion des comptes, des entreprises et l'accompagnement des étudiants.Administration globale : Gestion complète de la base de données et des droits utilisateurs.🛠️ Technique & SécuritéSécurité HTTPS : Configuration optimisée pour le protocole SSL (certificats auto-signés pour le développement local via XAMPP).Système de Cookies RGPD : Bannière de consentement personnalisable permettant d'activer ou de refuser les cookies statistiques.Design Responsive : Interface adaptable (Mobile First) avec menu burger fluide pour une navigation optimale sur tous les écrans.Architecture MVC : Séparation claire entre la logique (PHP Controllers/Models) et la présentation (Twig).🛠️ InstallationPrérequisXAMPP (ou tout autre serveur local Apache/MySQL).PHP 8.x ou supérieur.Composer (si vous souhaitez gérer des dépendances supplémentaires).Configuration LocaleCloner le dépôt :Bashgit clone https://github.com/ton-pseudo/stagepro.fr.git
Configuration Apache (HTTPS) :Activer le module SSL dans XAMPP.Pointer le DocumentRoot vers le dossier /public du projet.Base de données :Importer le fichier SQL (fourni en racine) dans PHPMyAdmin.Configurer les accès dans app/config/Database.php.🎨 Technologies utiliséesLangage : PHP 8.Template Engine : Twig.Frontend : HTML5, CSS3 (Custom Properties), JavaScript.Serveur : Apache via XAMPP.👥 CréditsDéveloppé avec passion par les étudiants de CESI.
